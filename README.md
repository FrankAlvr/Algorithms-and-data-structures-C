# Algorithms & Data Structures in C/C++

Collection of algorithm and data structure implementations in C and C++, solved as practice exercises on [LeetCode](https://leetcode.com).

Focused on building strong fundamentals in low-level programming — directly applicable to embedded systems and automotive software development.

---

## Topics covered

| Category | Problems |
|---|---|
| **Arrays** | Best time to buy and sell stock, Intersection of two arrays, Rotate array, Remove duplicates |
| **Strings** | Reverse string, Single number |
| **Bit manipulation** | Bit operations and masking (`ManejoBits.c`) |
| **Data structures** | Queue (circular buffer), Stack |
| **Math** | Reverse integer, Plus one |

---

## Languages

- **C** — primary language (`.c` files)
- **C++** — alternative solutions for selected problems (`.cpp` files)

---

## Structure

Each file is a self-contained solution with the problem name as the filename.

```
algorithms-and-data-structures-c/
│
├── Arrays/
│   ├── Best time to buy and sell stock.c
│   ├── Contains duplicate.c
│   ├── Intersection of two arrays.c
│   ├── Remove duplicates from sorted array.c
│   └── rotate array.c
│
├── Strings/
│   ├── Reverse string.c
│   └── Reverse string.cpp
│
├── BitManipulation/
│   └── ManejoBits.c
│
├── DataStructures/
│   └── Queue.c
│
└── Math/
    ├── Plus one.c
    ├── Reverse integer.c
    └── Single number.c
```

---

## Why C for algorithms?

Working through algorithmic problems in C builds the same low-level reasoning needed in embedded systems — manual memory awareness, pointer arithmetic, and efficient data manipulation without abstractions.
